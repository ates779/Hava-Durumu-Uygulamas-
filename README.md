🌤️ Modern Weather Dashboard

Python ile geliştirilmiş masaüstü bir hava durumu uygulamasıdır. Uygulama, OpenWeatherMap API üzerinden gerçek zamanlı hava durumu verilerini çekerek kullanıcıya sade ve hızlı bir arayüz üzerinden sunar.

         🚀 Özellikler
🌍 Gerçek zamanlı hava durumu verisi (OpenWeatherMap API)

⚡ Asenkron veri çekme ile akıcı kullanıcı deneyimi (aiohttp, asyncio)

📊 5 günlük hava durumu tahmini ve grafik gösterimi (matplotlib)

🗄️ Arama geçmişini kaydeden yerel SQLite veritabanı

🎨 Modern ve karanlık temalı kullanıcı arayüzü (CustomTkinter)

🔑 İlk çalıştırmada API anahtarı yapılandırma sistemi

🛠️ Kullanılan Teknolojiler
Python 3
aiohttp
asyncio
SQLite3
matplotlib
CustomTkinter
Pillow (PIL)

⚙️ Kurulum
Repoyu klonlayın:
git clone https://github.com/ates779/Weather-App.git
cd Weather-App
Gerekli paketleri yükleyin:
pip install -r requirements.txt
Uygulamayı çalıştırın:
python main.py

İlk çalıştırmada OpenWeatherMap API anahtarı girmeniz istenir. Ücretsiz olarak alabilirsiniz: https://openweathermap.org/

📌 Not

Bu proje, API kullanımı, asenkron programlama ve masaüstü uygulama geliştirme konularında pratik yapmak amacıyla geliştirilmiştir.
